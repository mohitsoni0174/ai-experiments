## 🗄️ Database Schema

### 4 Tables Created with RLS

**conversations**
- Tracks all chat sessions
- Stores intent and lead score
- References users

**messages**
- Individual messages
- Includes suggestions and metadata
- Tracks per-message intent

**leads**
- Qualified leads from chats
- Contact info + qualification data
- Status tracking (New→Converted)

**settings**
- Business configuration
- Bot settings
- Feature flags
