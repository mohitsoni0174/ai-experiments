## 📞 Troubleshooting

### Chat widget not showing?
```
1. Check browser console (F12)
2. Make sure JavaScript is enabled
3. Try hard refresh (Ctrl+Shift+R)
4. Check localhost:3000 loads
```

### Admin login failing?
```
1. Make sure you signed up first (/auth/signup)
2. Clear browser cookies
3. Check email/password spelling
4. Check console for errors
```

### Dev server not running?
```
1. Kill process: taskkill /F /IM node.exe
2. Reinstall: npm install
3. Start fresh: npm run dev
4. Check port 3000 is free
```

### Build failing?
```
1. Clear cache: rm -r .next
2. Reinstall: npm install
3. Try build: npm run build
4. Check console for errors
```
